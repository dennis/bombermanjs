bombermanjs
===========

Et lille Hello-World projekt, for at få lært javascript bedre at kende. Og så stifte bekendskab med NodeJS, Canvas og Socket.io. Det er spilbart, men ikke sjovt. Det er buggy og stadig ikke sjovt.
